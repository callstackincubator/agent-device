# Issue Tracker

This repo tracks work in GitHub Issues for `callstackincubator/agent-device`.

Engineering skills that create or update work items should use GitHub Issues and apply the triage labels documented in `docs/agents/triage-labels.md`.

Current environment note: this Codex workspace does not have the `gh` CLI installed, so issue publishing from this environment is blocked until `gh` is available and authenticated. When that is unavailable, keep PRDs or issue drafts under `.scratch/` and include the intended GitHub title and labels.
