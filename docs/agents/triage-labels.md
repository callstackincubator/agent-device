# Triage Labels

Use this default triage label vocabulary:

- `needs-triage`: maintainer needs to evaluate the issue.
- `needs-info`: waiting on reporter or requester.
- `ready-for-agent`: fully specified and ready for an AFK agent to pick up.
- `ready-for-human`: needs human implementation or decision-making.
- `wontfix`: will not be actioned.

PRDs produced by agent skills should use `ready-for-agent` once the work is sufficiently specified.
